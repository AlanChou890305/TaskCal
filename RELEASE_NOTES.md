# Release Notes - Version 1.3.0 (Build 28)

**Release Date**: 2026-03-06

---

## What's New

### English
- Terms of Use and Privacy Policy now open as smooth sheet modals instead of navigating to a separate screen
- Task creation and editing modal now shows a clear close (X) button and drag indicator for a more intuitive experience
- Improved visual consistency across modal sheets (headers, backgrounds, close buttons)
- Feedback form now shows a clearer validation message when the Details field is empty

### 繁體中文
- 使用條款與隱私權政策現以滑動 sheet modal 方式呈現，不再跳轉至獨立頁面
- 任務建立與編輯 modal 新增明顯的關閉（X）按鈕與拖曳指示條，操作更直覺
- 改善各 modal sheet 的視覺一致性（標題列、背景色、關閉按鈕）
- 回饋表單在「詳細內容」欄位為空時，現在顯示更清楚的提示訊息

### Español
- Los Términos de Uso y la Política de Privacidad ahora se abren como modales deslizables en lugar de navegar a una pantalla separada
- El modal de creación y edición de tareas ahora muestra un botón de cierre (X) claro y un indicador de arrastre para una experiencia más intuitiva
- Mejora de la consistencia visual en los modales (cabeceras, fondos, botones de cierre)
- El formulario de comentarios ahora muestra un mensaje de validación más claro cuando el campo Detalles está vacío
