## 測試離線/temp 任務的方法（dev build 不能直接飛航）
- **方法 B（推薦測 temp-delete）**：App 載入後開 Network Link Conditioner 高延遲（"Very Bad Network"），Metro 已載入仍可跑，Supabase 變慢 → temp 任務停留 → 趁空檔刪除
- **方法 C（送審前完整離線測）**：`npx expo run:ios --configuration Release --device`，bundle 打包進 App 不依賴 Metro，可完整測飛航模式
